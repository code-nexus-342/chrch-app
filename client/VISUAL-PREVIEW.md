## Typography

### Font Hierarchy
```
H1 (Titles)
Font: Inherit from Bootstrap
Size: 3.5rem (desktop), 2.5rem (mobile)
Weight: Bold (700)
Color: Gray-800 or White

H2 (Subtitles)
Size: 1.5rem (desktop), 1.25rem (mobile)
Weight: Normal (400)
Color: Gray-700 or Light Gray

Body Text
Size: 1.125rem (desktop), 1rem (mobile)
Weight: Normal (400)
Line Height: 1.75
Color: Gray-700 or Light Gray

Buttons
Size: 1rem
Weight: SemiBold (600)
Padding: 1rem 2rem
Border Radius: Full (rounded-full)
```

## Animation Examples

### Float Animation
```
     ●         ↑        ●
     ↑         ●        ↑
  ●  ●         ↓     ●  ●
     ↓      ●           ↓
  ●          ↑          ●
```

### Fade In Animation
```
Opacity: 0 ─────────────────────> 1
         0s                      0.8s

Elements fade in when scrolled into view
Duration: 800ms
Easing: ease-out
```

### Slide Up Animation
```
Position: +100px ──────────────> 0px
Opacity: 0 ────────────────────> 1
         0s                    0.6s

Content slides up as it appears
Duration: 600ms
Easing: ease-out
Stagger: 200ms between elements
```

## Responsive Breakpoints

### Mobile (< 768px)
```
- Single column layout
- Larger touch targets
- Simplified animations
- Smaller floating hub
- Stack cards vertically
```

### Tablet (768px - 1024px)
```
- 2-column grid where appropriate
- Medium spacing
- Full animations
- Standard floating hub
```

### Desktop (> 1024px)
```
- 3-column grid where appropriate
- Wider spacing
- All effects enabled
- Large floating hub
- Parallax backgrounds
```

## Accessibility Features

### Keyboard Navigation
```
Tab        → Move to next element
Shift+Tab  → Move to previous element
Enter      → Activate button/link
Space      → Scroll down
↑ ↓        → Scroll up/down
Escape     → Close modal/menu
```

### Focus Indicator
```
┌────────────────────────┐
│                        │ ← 2px solid gold outline
│    Focused Element     │   4px offset
│                        │
└────────────────────────────┘
```
